# 🍣 Japanese Recipe Explorer

A modern React-based web application that allows users to explore, search, and save Japanese recipes using external APIs.

---

## 🚀 Features

- 🍜 Browse Japanese recipes with pagination  
- 🔍 Search recipes using APIs  
- 🔽 Sort recipes (A–Z / Z–A)  
- 📄 View detailed recipe instructions and ingredients  
- ❤️ Add/remove favorites (stored in localStorage)  
- ⚡ Smooth and responsive UI  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6+)  
- CSS  
- React Router  
- Context API  

---

## 🔗 APIs Used

- TheMealDB API  
- Spoonacular API
